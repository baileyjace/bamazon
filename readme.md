# Skyrim Bamazon

A simple command line storeface straight out of Whiterun. 

## Technologies Used

The following languages and technologies were used:
* JavaScript
* node.js
* MySQL Workbench

The following npm packages were used:
* mysql
* inquirer
* cli-table
